# cmd_utils

Linux command line tools

## Setup

```yaml
dependencies:
  tekartik_cmd_utils:
    git:
      url: https://github.com/tekartik/cmd_utils.dart
      ref: dart2_3
    version: '>=0.2.0'
```
## kill_cmd

kill all chrome instance

    kill_cmd chrome
    
Kill qemu

    kill_cmd qemu
