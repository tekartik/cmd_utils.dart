# cmd_utils

Linux command line tools

## kill_cmd

kill all chrome instance

    kill_cmd chrome
    
Kill qemu

    kill_cmd qemu
    
## Activation

### From git repository

    pub global activate -s git git://github.com/tekartik/cmd_utils.dart

### From local path

    pub global activate -s path .
